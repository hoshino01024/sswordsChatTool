## 📝 项目介绍

用于中文版ppwords平台的ai聊天api调用网页.支持工具调用.导入导出聊天记录.调节模型温度.Max tokens和md等功能

## ✨ 主要功能

- 🤖 AI 聊天 API 调用
- 🔧 工具调用支持
- 💾 聊天记录导入/导出
- 🌡️ 模型温度调节
- 📊 Max tokens 配置
- 📝 Markdown 支持
